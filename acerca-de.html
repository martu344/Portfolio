Como trabajo.

Cuando empiezo a trabajar en un proyecto, trato de obtener toda la información y los datos de mi cliente que son relevantes para el proyecto, como objetivos, demografía y estética preferida. luego de esto empiezo a investigar sobre la industria, competencia, tendencias y demás factores que son necesarios para el desarrollo del proyecto.

cuando tengo todos los datos que necesito, mi siguiente paso es trabajar en la estructura alámbrica y el prototipo usando herramientas como adobe XD y figma. una vez que el prototipo está terminado y los wireframes están finalizados y aprobados, aquí es donde empiezo a hacer el diseño visual usando varias herramientas como adobe photoshop para procesar imágenes, adobe illustartor para crear gráficos vectoriales y adobe XD para construir el diseño visual real.

una vez que se aprueba el diseño visual, empiezo a codificar y transformar los diseños en código real utilizando una amplia gama de tecnologías como HTML, CSS y Javascript teniendo en cuenta las mejores prácticas
